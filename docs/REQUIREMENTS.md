# 项目需求文档

## 产品名称
通知栏便签 (Notification Notes)

## 核心功能
1. **通知栏便签**：文本内容直接显示在通知栏
2. **固定便签**（Sticky）：不被系统清除，重启后恢复
3. **定时提醒**：支持每日/每周/每月/自定义间隔
4. **快速添加**：悬浮窗输入、通知栏点击添加
5. **数据管理**：JSON 导出/导入，隐藏/恢复机制

## 详细需求

### 通知栏
- 每条便签独立通知线显示
- 蓝色半透明渐变背景，白色文字
- 格式：`内容 · 时间`
- 右侧显示 ✕ 隐藏按钮
- 点击通知 → 新建便签

### 便签管理
- 新增时设定是否固定到通知栏
- 从通知栏隐藏（数据保留）
- 应用内永久删除
- 编辑内容、切换固定状态
- 设置定时提醒

### 后台运行
- 前台服务保持通知持久化
- 开机自启（需授权）
- 电池优化白名单自动引导

### 导出
- 使用系统文件选择器（CreateDocument）
- JSON 格式，包含内容、创建时间
- 支持导入恢复
